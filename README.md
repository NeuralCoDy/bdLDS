# behavior_dLDS

This repository corresponds to the paper "Behavior-dLDS: A decomposed linear dynamical systems model for neural activity partially constrained by behavior"
by Eva Yezerets, En Yang, Misha B. Ahrens, and Adam S. Charles (https://openreview.net/forum?id=8p9tP9qLPN). The data can be found at https://osf.io/syvx5/.

Scripts for running the model can be found in the Experiments directory.
